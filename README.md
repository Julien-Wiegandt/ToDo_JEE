# ToDo_JEE
### The project
Here we have a ToDo list web app using JEE and Gradle frameworks, TomCat server and MySQL database.

Deployed on heroku -> https://todo-jee.herokuapp.com/

### Features
##### version 1.0
    * Login
    * Register
    * Edit Account settings
    * Delete Account
    * Add, Delete a TaskList
    * Add, Delete a Task
    * Deconnect
